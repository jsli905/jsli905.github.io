UAV Flight Control

Attitude estimation, control design, and autonomous quadcopter flight

Overview

This project focused on building a fully functioning quadcopter flight control system from the ground up. Rather than simply flying a drone, the goal was to develop the software that allows the vehicle to estimate its orientation, control its attitude, and complete a controlled flight task with minimal error.

The final demonstration required the quadcopter to take off autonomously, remain stable in the air, cross an obstacle, and land upright as close as possible to a target location.

What We Had To Do

  • Program and modify onboard flight software in C++
  • Identify the motor and propeller system (PWM → speed → thrust)
  • Build an attitude estimator using gyroscope and accelerometer data
  • Design and tune closed-loop attitude controllers
  • Implement hover control using sensor feedback
  • Integrate all subsystems into a fully working UAV

My Work

  • Modified C++ firmware to control motors and read sensor data
  • Implemented an attitude estimator using IMU measurements
  • Tuned controller parameters and tested stability through flight experiments
  • Performed system identification experiments for motor thrust and response
  • Debugged unstable behavior and improved flight smoothness
  • Helped complete the final demonstration flight

Results

By the end of the project, the quadcopter was able to take off autonomously, maintain stable flight using closed-loop control, cross an obstacle, and land upright with minimal position error. Performance was evaluated based on both landing accuracy and flight smoothness.
