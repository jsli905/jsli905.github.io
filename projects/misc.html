UAV Flight Control

Attitude estimation, control design, and autonomous quadcopter flight

Built a fully functioning quadcopter flight control system from the ground up from raw sensor data to stable autonomous flight. Developed software in C++ that allows the vehicle to estimate its orientation, control its attitude, and complete a controlled flight task with minimal error. In the end, the quadcopter could take off autonomously, remain stable in the air, cross an obstacle, and land upright as close as possible to a target location.

Overview

  • Developed onboard flight software in C++ to control motors and read sensor data
  • Identify the motor and propeller system (PWM → speed → thrust)
  • Build an attitude estimator using gyroscope and accelerometer data
  • Design and tune closed-loop attitude controllers and tested stability through flight experiments
  • Performed system identification experiments to relate motor input (PWM) to thrust and response
  • Tested and debugged unstable flight behavior through repeated flight experiments
  • Integrated estimation, control, and motor actuation into a fully working quadcopter

In the end, the quadcopter was able to take off autonomously, maintain stable flight using closed-loop control, cross an obstacle without touching anything, and land upright with minimal position error.
